# HTML 基本结构

HTML（超文本标记语言）是构建网页内容的基础语言，学习 HTML 是进入 Web 开发的第一步。本文将系统介绍 HTML 的基本结构与常用语法。

## HTML 的标准结构

```html

<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>我的第一个网页</title>
</head>
<body>
  <h1>Hello, world!</h1>
  <p>这是一个段落。</p>
</body>
</html>
```

### 各部分含义说明

| 标签/属性               | 描述                                                                                   | 作用与意义                                                                 |
|-------------------------|----------------------------------------------------------------------------------------|----------------------------------------------------------------------------|
| `<!DOCTYPE html>`       | 声明文档类型，告知浏览器这是一个 HTML5 文档。                                          | 确保浏览器以标准模式渲染页面，而不是怪异模式（Quirks Mode）。              |
| `<html lang="zh-CN">`   | HTML 文档的根元素，`lang="zh-CN"` 指定了文档的主要语言为简体中文。                     | 提供语义信息，帮助搜索引擎和屏幕阅读器更好地理解和处理内容。               |
| `<head>`                | 定义文档的头部，包含元信息（metadata），这些信息不会直接显示在网页上。                 | 提供关于文档的基本信息，如字符编码、标题、视口设置等。                     |
| `<meta charset="UTF-8">`| 定义文档的字符编码为 UTF-8，支持几乎所有语言的字符集。                                | 确保网页能够正确显示各种语言的字符，避免乱码问题。                        |
| `<meta name="viewport" content="width=device-width, initial-scale=1.0">` | 设置视口（viewport）的宽度为设备宽度，并将初始缩放比例设为 1.0。         | 优化移动端显示效果，使网页在不同设备上具有良好的响应式布局。              |
| `<title>`               | 定义网页的标题，显示在浏览器的标签页上。                                              | 提供网页的名称，帮助用户识别页面内容，也对 SEO（搜索引擎优化）有重要作用。 |
| `<body>`                | 定义文档的主体部分，包含所有可见的内容（如文本、图片、链接等）。                      | 包含网页的实际内容，是用户在浏览器中看到的部分。                          |
